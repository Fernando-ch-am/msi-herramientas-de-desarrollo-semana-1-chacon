## Objetivos del Trabajo
1. Aplicar conceptos fundamentales de control de versiones con Git y GitHub.
2. Utilizar el flujo de trabajo profesional integrado en Visual Studio Code.
3. Emplear el uso de ramas (feature branch) para nuevas funcionalidades.
4. Documentar el proceso de versionado de forma exhaustiva.

## Herramientas Utilizadas
* **Git:** Para el control de versiones local.
* **GitHub:** Para el alojamiento del repositorio remoto.
* **Visual Studio Code:** Como editor de código principal.
* **Git Graph:** Extensión de VS Code para visualizar el historial de ramas.
